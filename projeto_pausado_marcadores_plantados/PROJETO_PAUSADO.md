# ⏸️ PROJETO PAUSADO — Testbed de marcadores plantados

> **Status: PAUSADO em 2026-06-06** (não abandonado). Decisão tomada após conversa com o mestrando
> orientador: dado o **prazo de entrega** e as expectativas, o foco mudou para um **Plano B de fácil
> implementação** (ver `../plano_b_granada_pso/`). Este trabalho fica **arquivado e retomável**.

## O que é isto

A tentativa de construir um **otimizador multiobjetivo (HLY × Custo × Equidade)** validado por
**marcadores plantados** (verdade-base) sobre uma base sintética longitudinal de DM1. Inclui BFSS para
seleção de variáveis, simulador de HLY, política dinâmica por regras e a discussão do mFSS/MOFSS.

## Por que foi pausado

Ficou **complexo demais para o prazo**: base sintética longitudinal com marcadores, simulador de HLY,
política acoplada por orçamento, e a camada multiobjetivo (Pareto) sobre FSS misto. Bom como direção
de pesquisa, arriscado como entrega de curto prazo.

## Onde retomar

O documento canônico é **`ESTADO_ATUAL.md`** (nesta pasta) — ele tem todo o histórico de decisões
(escada de tratamento, M1/M2/M3, dano acumulado, epistemologia do BFSS, a §8 da possível mudança de
rota mFSS→discreto, etc.). É o ponto de partida para quem voltar a este plano.
